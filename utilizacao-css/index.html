oque e css

CSS (Cascading Style Sheets) é a linguagem de estilo responsável pela aparência de páginas web, permitindo controlar layout, cores, fontes, espaçamentos e animações. Ele separa a estruturação (HTML) da estilização, facilitando a criação de sites responsivos, reutilização de código e manutenção de designs complexos .



Funcionalidades

  • Layout: Controle de posicionamento usando Flexbox e Grid.
  • Responsividade: Uso de media queries para adaptar o layout a diferentes tamanhos de tela.
  • Estilização: Definição de cores, fontes, bordas, espaçamentos (padding/margin) e tamanhos.
